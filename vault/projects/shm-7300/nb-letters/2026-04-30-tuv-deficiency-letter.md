## Findings

### D1 — Stability evidence does not cover claimed shelf-life (Major)

**Section cited:** Technical File §4.2 (Stability)
**Reference:** IVDR Annex I §12.1; CLSI EP25-A
**Issue:** Real-time stability is documented to 9 months (STAB-003) against
a 24-month claim in the IFU. No accelerated stability data is provided.
**Evidence required:** Either extend the real-time study to T+24 months, or
add CLSI EP25-A-compliant accelerated stability with appropriate temperature
acceleration factor.

### D2 — Biocompatibility test report missing (Major)

**Section cited:** Technical File §4.5 (Biocompatibility)
**Reference:** ISO 10993-1 §6; ISO 10993-5
**Issue:** Patient-contacting material (PMMA sample chamber, MAT-002) has a
biological evaluation plan (BIO-001) but no completed cytotoxicity test
report. Plan alone is insufficient evidence.
**Evidence required:** Complete ISO 10993-5 cytotoxicity test report with
test sample preparation, controls, and acceptance-criteria justification.

### D3 — Risk control verification missing (Major)

**Section cited:** Risk Management File §7
**Reference:** ISO 14971:2019 §7.3
**Issue:** Hazard H-024 (aerosol carry-over) has a defined control measure
(closed-loop airflow) but no verification record. Existence of a control
measure is not equivalent to verification of its effectiveness.
**Evidence required:** Verification protocol VP-H024 demonstrating control
effectiveness under worst-case operating conditions.

### D4 — Symbols reference table absent from IFU (Minor)

**Section cited:** IFU §11
**Reference:** ISO 15223-1:2021 §5
**Issue:** Six symbols used correctly in the IFU; the standard cross-
reference table is not embedded.
**Evidence required:** Add ISO 15223-1:2021 reference table to the IFU.
