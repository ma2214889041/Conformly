# Interim Stability Report — SHM-7300

## 1. Scope

Real-time stability study of the SHM-7300 consumables (sample chambers,
aliquot tips, reagent reservoir) per CLSI EP25-A. Storage conditions:
25 ± 2 °C / 60 ± 5 % RH and 5 ± 3 °C.

## 2. Study design

- 3 manufacturing lots
- 3 replicates per timepoint per lot (n = 9 per timepoint)
- Timepoints: 0, 1, 3, 6, 9 months
- Acceptance criteria: ≤ 5 % drift in aliquot accuracy, no cross-contamination
  events, no visible material degradation

## 3. Results to date (T+9 months)

| Parameter | Spec | T+0 | T+3 | T+6 | T+9 | Within spec? |
|-----------|------|-----|-----|-----|-----|--------------|
| Aliquot CV @ 100 µL | ≤ 2 % | 1.3 % | 1.4 % | 1.5 % | 1.6 % | ✅ |
| Aliquot CV @ 500 µL | ≤ 2 % | 1.1 % | 1.2 % | 1.4 % | 1.5 % | ✅ |
| Carry-over events / 1000 cycles | 0 | 0 | 0 | 1 | 1 | ✅ |
| Visual degradation | None | none | none | none | none | ✅ |

## 4. Conclusion

The SHM-7300 consumables remain within specification through 9 months. No
evidence of material degradation. The study is ongoing through T+24 months
per the protocol STAB-001.

## 5. Gap declaration

⚠ This report **does not yet support** the IFU's 24-month shelf-life claim.
IVDR Annex I §12.1 requires evidence covering the full claimed lifetime.
Two options under evaluation:

1. Extend real-time study to T+24 months (target Apr 2027) and submit
   refreshed report. Blocks submission until then.
2. Add accelerated stability (CLSI EP25-A §6.2, Arrhenius / ASTM F1980)
   to bridge to T+24 months. Allows submission Q4 2026.

Recommendation pending RA leadership decision.
