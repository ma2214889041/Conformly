# Conformly

IVDR/CRO regulatory co-pilot for in-vitro diagnostic manufacturers entering the EU.

Built by [Firsteck Bio](https://firsteckbio.com) on top of the [NousResearch Hermes Agent](https://github.com/NousResearch/hermes-agent).

## Repo layout

- `plugin/` — Hermes plugin (Python tools)
- `vault/` — knowledge base (markdown, demo / mock data)
- `web/` — Next.js front-end (TBD, Day 4 of hackathon)
- `deploy/` — Vultr deployment scripts (TBD)
- `docs/` — design notes, demo prep
