# Firsteck Vault

这是 Firsteck Hermes Agent 的知识库,采用 Karpathy LLM-Wiki 架构。

## 目录结构

- `raw/` — 原始未整理资料(PDF、客户文档)
- `notes/` — Agent 整理过的结构化笔记
- `clients/` — 客户档案(每个客户一个 markdown)
- `projects/` — 项目工作日志
- `index/` — Agent 维护的导航索引

## 使用原则

1. `raw/` 文件夹是倾倒区,任何格式都可以
2. `notes/` 由 Agent 自动维护,人类不直接编辑
3. `clients/` 和 `projects/` 是 Agent 与顾问共同维护
4. 所有变更通过 git 追溯,审计友好
